**DocumentTemplate entity:**
```yaml
class: DocumentTemplate
fields:
  - name: string (required)
  - description: string (optional)
  - documentType: enum ['msa', 'nda', 'sow', 'compliance', 'other'] (required)
  - content: string (markdown with {{variables}}, required)
  - variableSchema: string (JSON array of custom variables, optional)
  - version: integer (starts at 1, incremented on edits, required)
  - status: enum ['draft', 'published', 'archived'] (default 'draft', required)
  - orgId: UUID (required, owner org)
  - createdBy: UUID (principal who created, required)
  - createdAt: timestamp
  - updatedAt: timestamp
properties:
  - No parent entity links (org-level, standalone)
  - Indexed on: orgId, status (for filtering templates by org and published status)
```

**DocumentInstance entity:**
```yaml
class: DocumentInstance
fields:
  - name: string (required, copied from template at creation)
  - description: string (optional)
  - documentType: enum ['msa', 'nda', 'sow', 'compliance', 'other'] (required, copied from template)
  - content: string (markdown with variables substituted, required)
  - templateId: UUID (source DocumentTemplate, required)
  - templateVersion: integer (version of template at instantiation time, required)
  - variableValues: string (JSON Record<string, string> of filled custom variables, optional)
  - engagementId: UUID (required, scope anchor)
  - projectId: UUID (optional, if scoped to project within engagement)
  - status: enum ['draft', 'final', 'signed'] (default 'draft', required)
  - createdAt: timestamp
  - updatedAt: timestamp
links:
  - linkTo: SmeMartProject.id.documentInstances (if projectId present)
  - Query by engagementId for listing on engagement detail
properties:
  - Indexed on: engagementId, projectId (for filtering by scope)
  - Duplicate prevention: unique constraint or query check on (templateId, engagementId, projectId)
```

}
