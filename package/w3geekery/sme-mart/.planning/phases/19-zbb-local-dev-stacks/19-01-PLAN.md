- `proxy_ssl_server_name on;` — SNI for uat.zerobias.com certificate
- `proxy_http_version 1.1` + `Upgrade`/`Connection` headers for WebSocket
- `include /etc/nginx/conf.d/apps/*.conf;` — app stacks inject their location blocks here (D-12)
- Use `${VAR}` for envsubst substitution; no nginx variables in those slots
