2. Create eslint.config.js at the project root (package.json level, not nested in src/app).

Use the complete skeleton from RESEARCH.md "eslint.config.js Skeleton" section (lines 190-317). Copy verbatim:
- Import statements (eslint, typescript-eslint, angular plugins)
- Global ignores block with node_modules, dist, .angular, .next

- Template files block with parser and plugins
  - Pattern 4: @angular-eslint/template/prefer-control-flow
  - Pattern 6: @angular-eslint/template/no-restricted-syntax (mat-spinner ban with selector "Element[name='mat-spinner']")
