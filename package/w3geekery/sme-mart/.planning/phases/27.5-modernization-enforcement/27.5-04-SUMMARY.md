## Pivot summary

The original Plan 04 spec called for annotating every pre-existing violation with `eslint-disable-next-line <rule> // pre-existing — pre-Phase-27.5 enforcement; track in MODERN-CLEANUP-1`. Probe on 2026-05-01 surfaced 1561 ESLint messages (vs. brief estimate ~250). Combined with Plan 03's pivot to diff-based CI lint, the annotation sweep became unnecessary: untouched files cannot fail CI regardless of disable comments. Annotating ~796 rule violations would impose a perpetual maintenance tax on every cleanup PR (remove disable comment + fix violation = double change surface).

Director resolution: replace annotation sweep with snapshot inventory + "touch it = fix it" rule (rule itself documented in Plan 05).
