## Key Questions Answered

### Discovery & Matching

- [x] **How do buyers find the right seller/freelancer?**
  - **Fiverr:** Browse categories → search → filter by budget, delivery time, seller level → view gig cards → gig detail page → seller profile
  - **Upwork:** Post job → receive proposals OR search talent → invite to apply → review proposals/profiles

- [x] **What filters matter most?**
  - **Fiverr:** Category, price range, delivery time, seller level, language, location
  - **Upwork:** Category/skills, hourly rate range, job success score, location, availability, talent badges

- [x] **How is quality/trust signaled?**
  - **Fiverr:** Seller levels (New→L1→L2→Top Rated), Pro badge, star rating (1-5), review count, response time, "Choice" badge
  - **Upwork:** Job Success Score (percentage), talent badges (Rising/Top Rated/Top Rated Plus/Expert-Vetted), earnings, hours worked, reviews

- [x] **How does search ranking work?**
  - **Fiverr:** Algorithm considers relevance, seller performance, response time, completion rate, reviews
  - **Upwork:** Relevance to job, profile completeness, JSS, activity, boosted proposals (paid)

### Engagement Setup

- [x] **How is scope defined?**
  - **Fiverr:** Gig packages define scope (deliverables, revisions, delivery time); Custom offers for bespoke
  - **Upwork:** Job post defines requirements; proposal includes cover letter, bid, milestones; contract formalizes

- [x] **How are terms agreed upon?**
  - **Fiverr:** Buyer selects package → requirements form → order starts. Custom: negotiate via inbox → custom offer
  - **Upwork:** Freelancer submits proposal with terms → client reviews → sends offer with final terms → freelancer accepts

- [x] **What's the minimum viable "contract"?**
  - **Fiverr:** Order created when buyer purchases gig package (implicit contract)
  - **Upwork:** Formal contract with project title, description, payment terms (hourly/fixed), milestones, weekly limit

- [x] **How is pricing communicated (fixed vs hourly)?**
  - **Fiverr:** Fixed packages displayed on gig (Basic $X / Standard $Y / Premium $Z); Hourly work available as separate feature
  - **Upwork:** Profile shows hourly rate; proposals can be hourly or fixed-price; contract type locked at creation

### Communication

- [x] **Where does messaging happen?**
  - **Fiverr:** Unified inbox for all conversations (pre-order inquiries, order messages, past clients)
  - **Upwork:** Messages workspace (per-contract rooms); Direct Messages (pre-contract, limited for free tier)

- [x] **Are there templates or prompts?**
  - **Fiverr:** Auto-reply feature for first-time messages; requirements form collects project details at order start
  - **Upwork:** No mandatory templates, but proposal form has structure (cover letter, bid, questions)

- [x] **How are response times tracked?**
  - **Fiverr:** Response time shown on profile ("Avg. response time: 1 hour"); affects search ranking
  - **Upwork:** Response rate tracked; aim for <2 hours recommended, 24 hours max

- [x] **File sharing capabilities?**
  - **Fiverr:** Up to 5GB per file in delivery; file attachments in messages
  - **Upwork:** File sharing in Messages; 500MB limit per file; integrations available

### Payment & Escrow

- [x] **When is money collected?**
  - **Fiverr:** At checkout before order starts (pre-pay)
  - **Upwork:** Fixed-price: at least first milestone funded before work; Hourly: charged weekly after work logged

- [x] **How does escrow work?**
  - **Fiverr:** Funds held by Fiverr until order complete; 14-day clearance period before available to seller
  - **Upwork:** "Project Funds" - client deposits → freelancer submits → 14 days to approve/request changes → auto-release if no action

- [x] **What are the fees (buyer/seller)?**
  - **Fiverr:** Buyer: 5.5% + $3.50 (orders <$200) | Seller: 20% commission
  - **Upwork:** Buyer: up to 7.99% + $0.99-$14.99 initiation | Seller: 0-15% variable per contract

- [x] **How are disputes handled?**
  - **Fiverr:** Resolution Center → freelancer has 48hrs to respond → if unresolved, CS reviews → Fiverr decides
  - **Upwork:** 7 days to dispute fixed-price → non-binding mediation (2 days for resolution) → binding arbitration if rejected

- [x] **Payment release triggers?**
  - **Fiverr:** Buyer accepts delivery OR 3 days auto-complete (8 days for milestones) + 14-day clearance
  - **Upwork:** Milestone: buyer approves OR 14 days no response; Hourly: weekly automatic based on logged time

### Work Tracking

- [x] **Time tracking tools?**
  - **Fiverr:** "Hourly Work" feature (newer) with hourly billing
  - **Upwork:** Desktop app with screenshots (random, several/hour), keystroke/mouse tracking, work diary

- [x] **Milestone management?**
  - **Fiverr:** Milestone orders available; 8 days to approve/request revision per milestone
  - **Upwork:** Create milestones at contract start or add during; each has deliverable, due date, budget; fund before work

- [x] **Deliverable submission?**
  - **Fiverr:** Deliver button → upload files (up to 5GB) → message to buyer → buyer reviews
  - **Upwork:** Submit Work for Review → client has 14 days to approve/request changes

- [x] **Revision handling?**
  - **Fiverr:** Number of revisions defined in package; buyer requests via order page
  - **Upwork:** Per milestone agreement; changes requested during 14-day review window

### Completion & Reviews

- [x] **How is work marked complete?**
  - **Fiverr:** Buyer accepts delivery OR auto-complete after 3 days (8 for milestones)
  - **Upwork:** Client approves milestone/releases payment; or ends contract

- [x] **What triggers review request?**
  - **Fiverr:** Order completion triggers review prompt for both parties
  - **Upwork:** Contract end; mid-contract feedback available upon request

- [x] **What's in the review form?**
  - **Fiverr:** Star rating (1-5), written review, optional photo/video, private satisfaction question
  - **Upwork:** Public: skills, availability, quality, deadlines, communication, cooperation (1-5 each), comments | Private: recommend score (0-10), end reason

- [x] **Can sellers respond to reviews?**
  - **Fiverr:** Yes, public response allowed
  - **Upwork:** Yes, can leave public response

- [x] **How are stats calculated?**
  - **Fiverr:** Success Score based on ratings, on-time delivery, cancellation rate, communication, disputes
  - **Upwork:** Job Success Score (JSS) from 6/12/24-month history; best score displayed; factors: public/private feedback, contract end reasons, disputes

## Platform-Specific Details

### Upwork: Connects System

- Connects are virtual tokens required to submit proposals
- Free tier: 10 Connects/month
- Plus plan: 100 Connects/month
- Connects required per job varies (based on project size, scope, demand)
- Boosted Proposals use extra Connects for visibility
- Connects refunded if client doesn't respond within 30 days

### Upwork: Contract Types

| Type | Description | Payment | Protection |
|------|-------------|---------|------------|
| **Hourly** | Pay by the hour, tracked via desktop app | Weekly automatic | Time logged in Work Diary |
| **Fixed-Price** | Pay per milestone | Upon milestone approval | Escrow/Project Funds |
| **Direct Contract** | Bring your own client | Reduced fees | Standard protection |

### Upwork: Talent Badges

| Badge | Requirements |
|-------|--------------|
| **Rising Talent** | New, 100% complete profile, verified ID, no negative feedback, active in 90 days |
| **Top Rated** | Top 10% of talent; 90%+ JSS for 13/16 weeks; consistent positive feedback |
| **Top Rated Plus** | Even higher bar; larger earnings history |
| **Expert-Vetted** | Upwork-reviewed and interviewed; top 1% |

### Fiverr: Seller Levels

| Level | Requirements |
|-------|--------------|
| **New Seller** | Just joined; no level yet |
| **Level 1** | Complete orders, maintain ratings, 60+ days active |
| **Level 2** | More orders, higher ratings, 120+ days |
| **Top Rated** | Exceptional performance, high volume, great reviews |
| **Pro** | Vetted by Fiverr; proven business experience; portfolio reviewed |

### Fiverr: Gig Packages

- 3-tier system: Basic / Standard / Premium
- Each tier defines: price, delivery time, revisions, features
- Minimum $5 (varies by category)
- "Gig Extras" can be added for additional cost
- Custom Offers bypass packages for bespoke work

## SME Mart Implications

### What to Adopt from Upwork

1. **Job Success Score concept** - Composite reputation metric visible on profile
2. **Milestone-based contracts** - Map to ZeroBias Tasks with subtasks
3. **Formal contract creation** - Auto-generate from request + accepted proposal
4. **Connects-like system** - Consider limiting free proposals to ensure quality (Phase 2+)
5. **Talent badges** - Rising, Established, Expert tiers based on ZB activity + reviews
6. **Work Diary / Time tracking** - Leverage ZB Tasks time tracking
7. **14-day review window** - Clear timeframe for deliverable approval

### What to Adopt from Fiverr

1. **Productized services** - Let providers create "gig-like" service offerings with packages
2. **3-tier pricing** - Basic/Standard/Premium structure for service offerings
3. **Response time display** - Show avg response time on provider profiles
4. **Seller levels** - Progressive unlock of features/visibility based on performance
5. **Resolution Center pattern** - Structured dispute resolution flow
6. **Auto-complete timer** - 3-day default, 8-day for complex work
7. **Success Score** - Holistic metric beyond just star rating

### What to Adapt

1. **Escrow → ZB Boundaries** - Use Boundary access as trust mechanism initially; true escrow Phase 3
2. **Messaging → ZB Task Comments** - All communication within task context for audit trail
3. **Contracts → ZB Tasks** - Task = contract; subtasks = milestones; comments = SOW details
4. **Verification → ZB Identity** - Platform identity verification; add skill verification badges
5. **Connects → Simplified** - Maybe free tier with limited proposals, paid for unlimited (Phase 2)

### What to Avoid

1. **20% seller fee** (Fiverr) - Too high for professional SME work; target 10-15%
2. **Screenshot surveillance** (Upwork hourly) - Trust-based for compliance professionals
3. **Complex fee structure** - Single clear fee, not multiple hidden fees
4. **Aggressive arbitration** - Human review first for compliance work context
