# Plan 050: Internal Marketplace (BU-to-BU)

**Status:** STUB
**Created:** 2026-03-13
**Source:** Marketplace meeting 2026-03-13 (Brian + Clark)

## Summary

SME Mart should support intra-company (business unit to business unit) marketplace scenarios using the same supply/demand constructs as external (company-to-company). Less legal overhead for internal, but same structural requirements (requirements, approvals, task tracking, billing).

## Key Concepts

- **Internal marketplace** — projects within a single parent company, one BU is demand, another is supply
- Same RFP/bid/engagement constructs apply
- Delineated differently from external (e.g., reduced legal/compliance requirements)
- A project could be "internal" or "external" — same UI, different flags/rules
- Brian's example: he (CEO) went internal to ZB team, proposed SME Mart as a project, Clark accepted as supply side
